How To Run:

1. Unzip, checkout, or clone the program files somewhere on your computer. 
2. Open Netbeans
3. In NetBeans go to File > Open Project...
4. Navigate to the program files on your computer. 
5. Netbeans should recognize the files as a project. Open it. 
6. Once open, go to the top of Netbeans and click the green arrow. 
7. The program should be running in a GUI at this point. If it's not, right click on PuzzleSolveFrame.java file in the "Projects" tab.
8. Enjoy.
